# מדריך למורים — איך מכינים ומעלים תרגיל חדש ב-EZRA

## שלב 1: הכינו את הגיליון

1. פתחו Google Sheets חדש.
2. בכל שורה: עמודה A = משפט באנגלית מלא, עמודה B = תרגום לעברית.
3. בלי כותרת, בלי עמודות נוספות, בלי משפטים עם קווים תחתונים (`___`) — המערכת יוצרת בעצמה את שלב ההשלמה בהמשך.

דוגמה:
| A | B |
|---|---|
| I like the blue T-shirt over there better than the red one. | אני אוהב את החולצה הכחולה שם יותר מהאדומה. |

## שלב 2: שתפו את הגיליון

בפינה הימנית העליונה → **Share** → שנו ל: **"כל מי שיש לו את הקישור — צפייה"** (Viewer).
בלי זה המערכת לא תוכל לקרוא את הגיליון.

## שלב 3: העתיקו את הקישור

פשוט מסרגל הכתובות של הדפדפן, בזמן שהגיליון פתוח.

## שלב 4: היכנסו ללוח המורה

https://ezra-demo-v2.onrender.com/teacher

בחרו את שמכם (דן / שרה) והזינו את סיסמת המורה שלכם.

## שלב 5: העלו את התרגיל

1. לשונית **"תרגילים"**.
2. כפתור **"⬇️ הוספת תרגיל חדש"**.
3. מלאו שם לתרגיל, הדביקו את הקישור מהגיליון.
4. לחצו **"הוסף ובחר תרגיל"**.

זהו — התרגיל נשמר במערכת ומופיע מיד לכל מורה, מכל מחשב.
